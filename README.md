# Kuaför Salonu Yönetim Uygulaması

Bu proje, ASP.NET Core MVC kullanarak geliştirilen bir kuaför/güzellik salonu yönetim sistemidir. Bayan ve erkek kuaför salonlarını yönetmek için bir platform sunar. Kullanıcılar, salonlar üzerinden randevu alabilir, çalışanlar hakkında bilgi edinebilir ve yapay zeka desteği ile saç modeli önerileri alabilirler.

## Proje Özellikleri

- **Salon Tanımlamaları**: Salonlar, sundukları işlemler, ücretler ve süreler detaylı olarak tanımlanabilir.
- **Çalışan Yönetimi**: Çalışanlar sisteme tanımlanabilir, uzmanlık alanları ve uygunluk saatleri belirlenebilir.
- **Randevu Sistemi**: Kullanıcılar, salon çalışanlarından randevu alabilir ve randevularının detaylarına ulaşabilirler.
- **REST API**: Veritabanı ile iletişim için REST API kullanılacak.
- **Yapay Zeka Entegrasyonu**: Kullanıcılar, fotoğraf yükleyerek saç modeli ve renk önerileri alabileceklerdir.

## Kullanılan Teknolojiler

- **ASP.NET Core 6 MVC**
- **C#**
- **SQL Server / PostgreSQL**
- **Entity Framework Core ORM**
- **Bootstrap**
- **HTML5, CSS3, JavaScript, jQuery**
- **Yapay Zeka API Entegrasyonu**

## Proje Hedefleri

Bu proje, web programlama dersinde öğrenilen bilgilerin pratiğe dökülmesi amacıyla geliştirilmiştir. Kullanıcı dostu bir arayüz ile salon yönetimini kolaylaştırmayı hedeflemektedir.
